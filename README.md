# BM Express — ระบบจัดการเงินและคำนวณกำไร

Web Application สำหรับบริษัทรับส่งพัสดุ ใช้แทนการบันทึกข้อมูลใน Excel 3 ไฟล์เดิม
พร้อมระบบคำนวณกำไรอัตโนมัติ, Dashboard สรุปยอดรายเดือน และ Export กลับเป็น Excel

## วิธีใช้งาน

**ดับเบิลคลิกไฟล์ `index.html`** — เปิดใช้งานได้ทันทีในเบราว์เซอร์ (Chrome / Edge)
ไม่ต้องติดตั้งโปรแกรมอะไรเพิ่ม

> แนะนำให้เปิดด้วยเบราว์เซอร์เดิมเสมอ เพราะข้อมูลถูกเก็บไว้ในเบราว์เซอร์เครื่องนั้น (localStorage)

## หน้าจอทั้งหมด

| หน้า | ใช้ทำอะไร |
|------|-----------|
| **ภาพรวม** | สรุปยอด: จำนวนพัสดุ, รายได้รวม, กำไรสุทธิ, เงินสดสุทธิ + กราฟเทรนด์รายเดือน + กรองได้หลายเดือนพร้อมกัน |
| **ข้อมูลตั้งต้น** | กำหนดรายการบรรจุภัณฑ์ ต้นทุน และราคาขาย (แก้ไข/ลบได้) |
| **รับพัสดุ / กำไร** | บันทึกพัสดุแต่ละชิ้น — เลือกบรรจุภัณฑ์แล้วระบบดึงราคาให้อัตโนมัติ พร้อมคำนวณกำไรสดๆ ขณะกรอก (แก้ไข/ลบได้) |
| **สรุปเงินสด** | เลือกวันที่ครั้งเดียว แล้วเพิ่มได้หลายรายการในวันนั้น แยกรายรับ/รายจ่าย-สินค้า/รายจ่าย-อื่นๆ (แก้ไข/ลบได้) |

## หน้าภาพรวม (Dashboard)

- **ตัวกรองเดือนแบบเลือกได้หลายเดือน** — คลิกเลือกได้ทีละหลายเดือนพร้อมกัน (ปุ่ม "ทั้งหมด" ล้างตัวกรอง) สถิติและกราฟจะรวมยอดของทุกเดือนที่เลือกไว้
- **กราฟเทรนด์รายเดือน** — แถบสีชมพูคือรายได้รวม แถบสีเขียว/แดงคือกำไร (แยกตามบวก/ลบ) ของแต่ละเดือน แสดงทุกเดือนที่มีข้อมูลเสมอ ไม่ขึ้นกับตัวกรอง

## หน้าสรุปเงินสด — บันทึกทีละวัน หลายรายการ

กดปุ่ม "+ บันทึกรายการ" แล้วเลือกวันที่เพียงครั้งเดียว จากนั้นกด **"+ เพิ่มรายการ"** เพื่อเติมรายการรับ-จ่ายอื่นๆ ของวันเดียวกันได้เรื่อยๆ ในฟอร์มเดียว (ลบรายการที่เพิ่มเกินได้ด้วยปุ่ม ×) กด "บันทึก" ครั้งเดียวจะแยกเก็บเป็นรายการย่อยตามปกติ — ตรงกับวิธีที่สมุดบันทึกเดิมทำ (1 วัน มีได้หลายบรรทัด)

## สูตรคำนวณกำไร

```
กำไรสุทธิ = รายได้รวม − (ต้นทุนกล่อง + ต้นทุนขนส่ง + ต้นทุนใบปะหน้า)
```

> ค่าบริการ**ไม่ถูกหัก**เป็นต้นทุน เพราะเป็นรายรับที่รวมอยู่ในรายได้แล้ว (ไม่มีต้นทุน)
> — ตรงตามวิธีคิดในไฟล์ `กำไรบริษัท BM Express.xls` เดิม

## ข้อมูลเก่าที่นำเข้าแล้ว (seed-data.js)

เปิดแอปครั้งแรกจะมีข้อมูลเก่าจาก Excel ครบทั้งหมด:

- **บรรจุภัณฑ์ 12 รายการ** จาก `รายการและต้นทุน.xls` (+ "ลูกค้าแพ็คสินค้ามาเอง")
- **พัสดุ 384 รายการ** จาก `กำไรบริษัท BM Express.xls` (เทอม 1-2568 และ 2-2568)
- **เงินสดรับ-จ่าย 318 รายการ** จาก `income-expenses.xls` (ปี 2567, เทอม 1-2568, 2-2568)

หมายเหตุการนำเข้า:
- แถว "สรุป" รายวัน, "ยอดเงินรวม", "ยกยอด" ไม่ถูกนำเข้า — ระบบคำนวณเองจากรายการจริง
- ชื่อบรรจุภัณฑ์ที่สะกดหลายแบบ (เช่น กล่องB / กล่อง B / กล่องเบอร์ B) ถูกรวมเป็นชื่อเดียว
- รายการ "รายได้เป็นเงินสด" ที่กรอกไว้คอลัมน์ขวาในไฟล์เดิม ถูกนับเป็นรายรับ (13 รายการ)

### รายการปรับยอด (สำคัญ)

ยอดสุทธิที่คำนวณจากรายการจริงในไฟล์เดิมทั้งหมดคือ **35,244.82 บาท** แต่สมุดบันทึกเดิมลงท้ายไว้ที่ 28,243.75 บาท
ส่วนต่าง 7,001.07 บาท มาจาก 3 จุดในไฟล์เดิม (สูตร SUM ตกหล่นข้อมูลบางช่วง + รายการอยู่นอกช่วงสูตร + ยกยอดพิมพ์ผิด)
มีรายการ **"ปรับยอดให้ตรงกับเงินสดที่นับได้จริง"** (รายจ่าย-อื่นๆ 7,001.07 บาท ลงวันที่เปิดใช้แอป) เพื่อให้ยอดสุทธิในแอปตรงกับเงินสดที่นับได้จริง 28,243.75 บาท — แก้ไข/ลบรายการนี้ได้ในหน้าสรุปเงินสดหากต้องการปรับใหม่

## Export Excel

ปุ่ม **Export Excel (3 ไฟล์)** ที่มุมล่างซ้าย จะสร้างไฟล์ `BM_Express_Export.xlsx` ที่มี 3 sheets:

1. **รายการและต้นทุน** — ข้อมูลตั้งต้นบรรจุภัณฑ์
2. **กำไรบริษัท BM Express** — รายการพัสดุทั้งหมดพร้อมกำไร
3. **Income-Expenses** — สรุปเงินสดรับ-จ่าย

*(การ Export ต้องต่ออินเทอร์เน็ตครั้งแรกเพื่อโหลด library — หลังจากนั้นเบราว์เซอร์จะ cache ไว้)*

## การเก็บข้อมูล & สำรองข้อมูล

- ข้อมูลถูกเก็บใน **localStorage** ของเบราว์เซอร์ (key: `bmexpress_v1`) — ไม่หายเมื่อปิดเบราว์เซอร์
- ⚠️ ข้อมูลจะหายถ้าล้าง "ข้อมูลการท่องเว็บ" (Clear browsing data) ของเบราว์เซอร์
- **แนะนำให้กด Export Excel เก็บไว้เป็นประจำ** เพื่อสำรองข้อมูล

## ไฟล์ในโปรเจกต์

- `index.html` — โครงหน้าเว็บทั้งหมด (4 หน้า + 3 modal)
- `style.css` — สไตล์ตาม design ต้นแบบ (โทนชมพู #e91e8c)
- `app.js` — logic ทั้งหมด: state, คำนวณกำไร, กรองเดือน, export Excel
- `README.md` — เอกสารนี้
